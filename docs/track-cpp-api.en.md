## Reflection macro guide

<div class="api-macro-grid">
<article><code>UCLASS</code><p>Registers a UObject-derived class with Unreal reflection. Class metadata controls how the component appears in the editor.</p></article>
<article><code>USTRUCT</code><p>Registers a reflected value type. BlueprintType makes the struct usable as a Blueprint pin and variable type.</p></article>
<article><code>UPROPERTY</code><p>Registers a field for serialization, editor exposure, garbage collection, replication, or Blueprint access according to its specifiers.</p></article>
<article><code>UFUNCTION</code><p>Registers a callable or query function. BlueprintCallable, BlueprintPure, and CallInEditor define where it is exposed.</p></article>
<article><code>GENERATED_BODY</code><p>Injects the code generated by Unreal Header Tool. It must remain inside every reflected class or struct.</p></article>
<article><code>EditAnywhere / VisibleAnywhere</code><p>Controls whether a property is editable or only visible in Details.</p></article>
<article><code>BlueprintReadWrite / BlueprintReadOnly</code><p>Controls whether Blueprint graphs can write the property or only read it.</p></article>
<article><code>Category / meta</code><p>Organizes the editor UI and supplies display names, tooltips, conditions, units, clamps, and advanced-display rules.</p></article>
</div>
