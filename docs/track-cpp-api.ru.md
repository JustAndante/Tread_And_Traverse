## Макросы отражения

<div class="api-macro-grid">
<article><code>UCLASS</code><p>Регистрирует класс-наследник UObject в системе отражения Unreal. Метаданные класса определяют его отображение в редакторе.</p></article>
<article><code>USTRUCT</code><p>Регистрирует отражаемый тип-значение. BlueprintType разрешает использовать структуру в переменных и пинах Blueprint.</p></article>
<article><code>UPROPERTY</code><p>Регистрирует поле для сериализации, редактора, сборщика мусора, репликации или Blueprint — в зависимости от спецификаторов.</p></article>
<article><code>UFUNCTION</code><p>Регистрирует функцию или запрос. BlueprintCallable, BlueprintPure и CallInEditor определяют, где она будет доступна.</p></article>
<article><code>GENERATED_BODY</code><p>Подключает код, созданный Unreal Header Tool. Макрос должен оставаться внутри каждого отражаемого класса или структуры.</p></article>
<article><code>EditAnywhere / VisibleAnywhere</code><p>Определяет, можно ли менять свойство в Details или оно доступно только для просмотра.</p></article>
<article><code>BlueprintReadWrite / BlueprintReadOnly</code><p>Разрешает Blueprint-графам изменять свойство либо только читать его.</p></article>
<article><code>Category / meta</code><p>Задаёт раздел интерфейса, отображаемое имя, подсказку, условия показа, единицы, ограничения и Advanced Display.</p></article>
</div>
